2. Day 0A — Python Foundations Refresher

2.1 Variables and Functions: State vs. Behavior

A variable assigns a symbolic label to a memory location holding a specific value. A function encapsulates a reusable sequence of computations (a behavior) that accepts inputs, executes a contract, and explicitly returns an output. 

# Variable assignment: Storing a scalar value
capacity_kw = 250.0

# Function definition: Defining an operational contract
def kw_to_mw(capacity_kw: float) -> float:
    '''Convert electrical capacity from kilowatts (kW) to megawatts (MW).'''
    return capacity_kw / 1000.0

# Calling the function
result_mw = kw_to_mw(capacity_kw)
print(result_mw)  # Output: 0.25

Line-by-Line Breakdown:

  • def kw_to_mw(capacity_kw: float) -> float: — def tells Python we are defining a new function. capacity_kw is the parameter. : float and -> float are type hints.
  • '''Convert electrical capacity...''' — The docstring, accessible via help(kw_to_mw).
  • return capacity_kw / 1000.0 — Explicitly returns the result. Omitting return defaults to returning None.

2.2 Loops and List Iteration

A for loop iterates over a sequential collection (such as a list), binding each element to a loop variable one step at a time. 

capacities_kw = [100.0, 250.0, 500.0, 1800.0, 60.0]

capacities_mw = []
for c in capacities_kw:
    capacities_mw.append(kw_to_mw(c))

print(capacities_mw)  # Output: [0.1, 0.25, 0.5, 1.8, 0.06]

2.3 List Comprehensions: Syntactic Unrolling

A list comprehension is a concise way to create lists. It performs the exact same operation as a standard for loop with .append(), but in a single line. 

# Syntax: [expression for item in iterable]
capacities_mw = [kw_to_mw(c) for c in capacities_kw]

2.4 Dictionaries: Label-Value Mapping

A dictionary (dict) stores data using key-value pairs, maintaining explicit labels for every value instead of relying on positional indices. 

project = {
    "site_id": "MHP-0042",
    "cable_length_km": 15.0,
    "planned_capacity_kw": 100.0,
    "terrain_index": 3,
}

print(project["planned_capacity_kw"])  # Output: 100.0

2.5 Object-Oriented Encapsulation: The self Keyword

A class is a blueprint that combines state (data attributes) and behavior (methods) into a single object. 

class UnitConverter:
    def __init__(self, factor: float):
        self.factor = factor  # Store factor inside this specific instance

    def convert(self, value: float) -> float:
        return value * self.factor

kw_to_mw_converter = UnitConverter(factor=0.001)
print(kw_to_mw_converter.convert(250.0))  # Output: 0.25
The Standard List Multiplication Trap

In standard Python, [100.0, 250.0] * 2 produces [100.0, 250.0, 100.0, 250.0] (list repetition), NOT [200.0, 500.0]. Vectorized scaling requires NumPy arrays.

3. Day 0B — NumPy, Array Shapes, and Matrix Algebra

3.1 Why NumPy? Contiguous Memory and Vectorization

Standard Python lists store pointers to scattered objects. A NumPy array (np.ndarray) stores numerical values in contiguous memory blocks, enabling vectorized CPU SIMD execution. 

import numpy as np

capacities_kw = np.array([100.0, 250.0, 500.0])
capacities_watts = capacities_kw * 1000.0
print(capacities_watts)  # [100000. 250000. 500000.]

3.2 Array Dimensions and the .shape Attribute

a = np.array([1.0, 2.0, 3.0])        # Shape: (3,)   -> 1D Vector
b = np.array([[1.0, 2.0, 3.0]])      # Shape: (1, 3) -> 2D Row Matrix
c = np.array([[1.0], [2.0], [3.0]])  # Shape: (3, 1) -> 2D Column Matrix
Array Syntax .shape Dimensions (.ndim) Conceptual Meaning
np.array([1, 2, 3]) (3,) 1 Flat 1D Vector
np.array([[1, 2, 3]]) (1, 3) 2 2D Row Matrix (1 Row, 3 Cols)
np.array([[1], [2], [3]]) (3, 1) 2 2D Column Matrix (3 Rows, 1 Col)

3.3 Matrix Slicing & Indexing

X = np.array([
    [15.0, 100.0, 3.0],   # Row 0
    [30.0, 250.0, 4.0],   # Row 1
    [5.0,  500.0, 1.0],   # Row 2
])

print(X[0, :])   # First row: array([15., 100., 3.])
print(X[:, 0])   # First column: array([15., 30., 5.])
print(X[:2, :2]) # First 2 rows, first 2 columns

3.4 Broadcasting Mechanics

NumPy compares array dimensions from right to left. Dimensions match if they are equal or if one of them is 1. 

X = np.array([
    [15.0, 100.0],
    [30.0, 250.0],
    [5.0,  500.0]
])  # Shape: (3, 2)

means = np.array([16.7, 283.3])  # Shape: (2,)

centred = X - means  # Subtracts column means from every row

3.5 The Dot Product at Three Abstraction Levels

a · b = Σi=1p ai bi = a1 b1 + a2 b2 + ... + ap bp 
features = np.array([15.0, 100.0])
weights  = np.array([0.05, 0.002])

# Level 1: Scalar Expansion
manual = features[0] * weights[0] + features[1] * weights[1]

# Level 2: Python Loop
loop_total = sum(f * w for f, w in zip(features, weights))

# Level 3: Vectorized NumPy Operator (@)
dot_total = features @ weights

print(manual, loop_total, dot_total)  # Output: 0.95 0.95 0.95

4. Day 0C — Algebra, Summation Notation, and Loss Functions

4.1 Equations of Lines & Linear Models

y = m x + c   &implies;   ŷ = β0 + β1 x1 + β2 x2 + ... + βp xp

4.2 Summation Notation (Σ) and SSR

SSR = Σi=1n (yi - ŷi)2 
y_actual    = np.array([12.0, 30.0, 8.0])
y_predicted = np.array([11.2, 31.5, 9.0])

# Explicit Loop
ssr_loop = sum((a - p) ** 2 for a, p in zip(y_actual, y_predicted))

# Vectorized NumPy One-Liner
ssr_numpy = np.sum((y_actual - y_predicted) ** 2)

print(ssr_loop, ssr_numpy)  # Output: 2.89 2.89

4.3 Critical Distinction: Sum of Squares vs. Square of Sum

Σ xi2 ≠ (Σ xi)2

For x = [2.0, -3.0, 5.0]:
• Sum of Squares: 22 + (-3)2 + 52 = 4 + 9 + 25 = 38.0
• Square of Sum: (2 - 3 + 5)2 = 42 = 16.0

5. Day 0D — First-Principles Statistics

Statistic Formula Python Implementation
Mean (x&bar;) 1/n Σ xi np.sum(x) / len(x) or x.mean()
Variance (σ2) 1/n Σ (xi - x&bar;)2 np.sum((x - x.mean())**2) / len(x)
Std Dev (σ) √(σ2) np.sqrt(x.var()) or x.std()
Covariance 1/n Σ (xi - x&bar;)(yi - y&bar;) np.sum((x - x.mean())*(y - y.mean())) / len(x)
Correlation (r) cov(x,y) / (σx σy) cov_xy / (x.std() * y.std())
Proof that Raw Deviations Sum to Zero
Σ (xi - x&bar;) = Σ xi - Σ x&bar; = n x&bar; - n x&bar; = 0

Because raw positive and negative deviations always sum to zero, we must square deviations before taking their average.

6. Day 0E — Calculus, Gradients & Gradient Descent

6.1 Limit Definition of Derivative

f'(x) = limh → 0 [f(x + h) - f(x)] / h 
def f(x): return x ** 2
def numerical_slope(f, x, h=1e-6):
    return (f(x + h) - f(x)) / h

print(numerical_slope(f, 3.0))  # Output: ~6.000001

6.2 Partial Derivatives & Gradient Descent

∇ E(m, c) = [ ∂E/∂m, ∂E/∂c ]T   &implies;   θnew = θold - α ∇ E(θ) 
def error(m, c): return (10.0 - (m * 2.0 + c)) ** 2
def partial_m(m, c, h=1e-6): return (error(m + h, c) - error(m, c)) / h
def partial_c(m, c, h=1e-6): return (error(m, c + h) - error(m, c)) / h

m, c, lr = 0.0, 0.0, 0.01
for _ in range(50):
    m -= lr * partial_m(m, c)
    c -= lr * partial_c(m, c)

print(f"Optimized: m={m:.3f}, c={c:.3f}")

7. Level 0 Capstone Complete Reference Solution

import numpy as np

projects = np.array([
    [12.0, 15.0],
    [30.0, 25.0],
    [5.0,  8.0],
    [40.0, 45.0],
    [15.0, 12.0]
])
costs = np.array([12.0, 30.0, 8.0, 45.0, 15.0])
n_obs = len(projects)

# 1. Shape Inspection
print("1. Shape:", projects.shape)  # (5, 2)

# 2. Custom Loop Means & Stds
means, stds = [], []
for j in range(projects.shape[1]):
    col = projects[:, j]
    m = sum(col) / n_obs
    s = np.sqrt(sum((x - m)**2 for x in col) / n_obs)
    means.append(m)
    stds.append(s)

print("2. Means match:", np.allclose(means, projects.mean(axis=0)))
print("   Stds match :", np.allclose(stds, projects.std(axis=0)))

# 3. Pearson Correlation
x = projects[:, 0]
cov_xy = sum((x[i] - means[0]) * (costs[i] - costs.mean()) for i in range(n_obs)) / n_obs
r_manual = cov_xy / (stds[0] * costs.std())
print(f"3. Correlation r = {r_manual:.4f}")

# 4. Matrix Prediction
beta = np.array([0.5, 0.2])
pred_loop = np.array([sum(projects[i] * beta) for i in range(n_obs)])
pred_matrix = projects @ beta
print("4. Predictions match:", np.allclose(pred_loop, pred_matrix))

# 5. SSE
def sse(y_true, y_pred): return np.sum((y_true - y_pred) ** 2)
current_sse = sse(costs, pred_matrix)
print(f"5. Current SSE: {current_sse:.4f}")

# 6. Finite Difference Derivative
h = 1e-5
beta_nudged = beta.copy()
beta_nudged[0] += h
nudged_sse = sse(costs, projects @ beta_nudged)
d_sse_dbeta0 = (nudged_sse - current_sse) / h
print(f"6. d(SSE)/d(beta_0) = {d_sse_dbeta0:.4f}")

# 7. Intentional Shape Error
try:
    bad_beta = np.array([0.5, 0.2, 0.9])
    invalid_pred = projects @ bad_beta
except ValueError as e:
    print("7. Caught expected error:", e)
