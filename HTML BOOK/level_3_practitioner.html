11. Data Preprocessing

11.1 Numeric Scaling

Linear models with regularization and SVMs require scaling. Otherwise, features with larger magnitudes dominate the coefficients.

  • StandardScaler: Center to mean 0, variance 1. Good default.
  • MinMaxScaler: Bounded scale between 0 and 1.
  • RobustScaler: Uses median and IQR. Ideal when data has extreme outliers.

11.2 Categorical Encoding

Convert text labels into model-ready numbers.

  • One-Hot Encoding: Creates binary dummy variables. Drop the first column to prevent collinearity.
  • Target Encoding: Replaces categories with target mean. Risk of severe data leak unless managed inside CV loop.

12. Feature Engineering

Domain-specific adjustments to improve linear models.

Interactions

Multiply features when their effects are interdependent (e.g. fertilizer $\times$ irrigation).

Polynomials

Add squared terms ($X^2$) to fit simple curves like optimal soil pH ranges.

Ratios

Calculate ratios like public funding per capita to represent scale-free variables.

13. Linear Regression in Depth

13.1 Coefficients and Reference Categories

When interpreting coefficients, remember: continuous variables represent the change in target per 1-unit increase, holding all other features constant. Dummy variables show the difference relative to the dropped reference class.

13.2 Variance Inflation Factor (VIF)

Measures how much the variance of an OLS coefficient is inflated by multicollinearity. Calculated as $1/(1 - R_j^2)$. VIF values above 10 indicate severe collinearity that renders parameter significance estimates unstable.

14. Assumptions and Diagnostics

14.1 Heteroscedasticity

Heteroscedasticity occurs when residual variance is not constant across fitted predictions (visualized as a funnel shape). To fix it, log-transform the target, use weighted least squares, or apply robust standard errors like HC3 in statsmodels.

15. Regularized Linear Models

Ridge (L2)

Adds squared penalty ($\alpha \sum \beta_j^2$). Shrinks weights but keeps all features. Best when dealing with severe multicollinearity.

Lasso (L1)

Adds absolute value penalty ($\alpha \sum |\beta_j|$). Can force weights to exactly 0, performing automatic feature selection.

ElasticNet

Combines L1 and L2 penalties using l1_ratio. Best of both worlds for sparse systems with correlated inputs.
